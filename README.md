# MethodsNET Advanced Text Analysis Summer School

This repository contains the code materials for the five-day advanced text analysis 2026 summer school.

## Student Quick Start

From the repository root, run:

```bash
python3 install_dependencies.py
```

On Windows, use:

```bash
python install_dependencies.py
```

Then start Jupyter:

```bash
source .venv/bin/activate
jupyter lab materials/notebooks
```

On Windows:

```bash
.venv\Scripts\activate
jupyter lab materials/notebooks
```
